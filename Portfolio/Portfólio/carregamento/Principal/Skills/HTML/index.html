Tech User

A Tech User é um projeto de desenvolvimento web , que , vai da etapa de planejamento e design ,que foi feita no Figma até a etapa de desenvolvimento, feita com Html , Css e Js, sem uso de frameworks.

Qual o intuito da Tech User ?
• Listar peças de hardware e ranqueá-las.
• Ajudar usuários casuais a terem noção básica de Hardware
Observação : é válido lembrar que este projeto ainda está em fase desenvolvimento, por este motivo , o resultado final pode sofrer alterações diversas.

Sou um aspirante a desenvolvedor front-end com experiência intermediária
em HTML5 e CSS3, além de conhecimentos práticos na implementação de
JavaScript em projetos.
